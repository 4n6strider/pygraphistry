Histograms!




Histograms


The histogram panel is one of the most powerful tools in our UI, and though it might be intimidating at first, it’s actually quite simple. You can create a histogram that will look at any attribute of the elements in your graph, this helps you get an idea about how the data in your graph is divided. Lets look at an example…






Create A Histogram

You can create a histogram that will look at any attribute of the elements in your graph, this helps you get an idea about how the data in your graph is divided.

For instance, if we wanted to look at point:degree we can see a representation of all the nodes in our graph, broken down into buckets based on their degree. We can see here that we have a lot of nodes with low connectivity and only a small amount of nodes with a very high connectivity.



Select A Range To Create Filters

You can create filters on the fly using histograms, simply click one of the buckets, or click and drag multilpe buckets and see our visualization pare down to nodes with attributes within that range. Click the selected bucket range again to delete the filter.



We just created this cool filter!



Histogram Settings

Lets jump into this histograms’ advanced settings, so we can apply a color scheme for our graph.



Applying A Color Profile

Say we wanted to be able to see from far away which elements in our graph have the most connections? In our histogram settings, we can select a gradient color scheme, and now we can see our highly connected nodes in red, and our lowe connectivity nodes in blue.



Result

We just used a histogram to apply a custom color scheme to an attribute we we’re interested in. Try playing with some different attribute histograms!
