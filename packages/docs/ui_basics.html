Filters & Exclusions




Filters


Using filters is a great way to prune down a large graph to just see elements that you are interested in exploring, in this example we'll look at how we casn use a filter to only look at the most highly connected characters within our Les Mis graph.






Filter Example

So if we want to create a filter for looking at only the most highly connected nodes, we need to choose which attribute we want to filter by, in this case we use the attribute Degree to describe the # of incoming and outgoing edges of a single node. Both nodes and edges have attributes so we'll use the selector point: instead of edge:.



1. Select An Attribute

So let’s open the filter panel and find point:degree(node:attribute) to create a new filter



2. Enter A Value

We want to only look at nodes with a degree value of over 15, we’ll set our filter value to…

point:degree >= 15



Result

Now we can see that we only have a handful of nodes, and we can recluster (using the play button) the graph to see connections.




Exclusions


Exclusions work very similary to filters, except now we're choosing which elements of our graph we'd like to take out. In this example we'll exclude all of the highly connected nodes in our graph and see what remains.






1. Select An Attribute

So let’s open the exclusion panel and find point:degree again to create a new exclusion



2. Enter A Value

This time we want to take out all the nodes with a degree of connection higher than 5, so our value will be...

point:degree >= 5



Result

Now we can see all of the least connected nodes in our graph, as we've excluded all of the nodes with a degree of connection higher than 5.
